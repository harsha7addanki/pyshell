# Requirements
* Your Device Needs The Latest Version Python

# How To Install

Step #1:
open your terminal in the directory that `psh-inst.py` is
NOTE: To Get `psh-inst.py` download the `psh-installer` Branch

Step #2:
run `psh-inst.py`:

linux:
```bash
pip3 install tkinter
pip3 install tk
pip3 install ttk
python3 psh-inst.py
```
windows:
```batch
python psh-inst.py
```
mac:
```bash
python psh-inst.py
```
